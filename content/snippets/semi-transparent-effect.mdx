---
title: 'Semi-Transparent Effect'
description: 'How to make a box semi-transparent.'
---

```css
/* white bg (#fff), 80% opacity */
.semi-transparent-hsla {
  background-color: hsla(0, 0%, 100%, 0.8);
}
.semi-transparent-rgba {
  background-color: rgba(255, 255, 255, 0.8);
}
```
