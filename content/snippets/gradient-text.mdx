---
title: 'Gradient Text'
description: 'Set Gradient Colors for Text with CSS.'
category: 'css'
---

```css {6-13}
.gradient-text {
  --gradient-direction: to right;
  --gradient-from: #5237f9;
  --gradient-to: #00b8b9;
  --gradient-stops: var(--gradient-from), var(--gradient-to);
  display: inline;
  background-image: linear-gradient(
    var(--gradient-direction),
    var(--gradient-stops)
  );
  -webkit-text-fill-color: transparent;
  -webkit-background-clip: text;
  background-clip: text;
}
```

## Result

<GradientText variant="winter" textSize="2xl">
  Winter is Coming.
</GradientText>
