## Tech

- [React](https://reactjs.org) — The most widely used Front-End Framework.
- [NextJS](https://nextjs.org) — The best React Framework.
- [TypeScript](https://www.typescriptlang.org) — Catch errors early in your editor. A must have for JavaScript projects.
- [Prettier](https://prettier.io/) — Enforces a consistent code style across your entire codebase.
- [Tailwind CSS](https://tailwindcss.com) — A great way to keep my styling consistent and stay productive.
- [Vercel](https://vercel.com) — I host most of my websites here. _Fast is the name of the game_.
- [Github](https://github.com) — Where I host my code.
- [Figma](https://www.figma.com) — My preferred design tool.
- [Radix UI](https://www.radix-ui.com) — An accessibility-focused set of easy-to-use and easy-to-style components.
