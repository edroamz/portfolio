---
title: 'Gradient Text'
description: 'Set Gradient Colors for Text with CSS'
---

<Callout>Element should have an `inline` property or similar.</Callout>

```css {7-12}
.gradient-text {
  --gradient-direction: to right;
  --gradient-from: #e36b52;
  --gradient-via: #632cda;
  --gradient-to: #5237f9;
  --gradient-stops: var(--gradient-from), var(--gradient-via), var(--gradient-to);
  background-image: linear-gradient(
    var(--gradient-direction),
    var(--gradient-stops)
  );
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

Result:

<GradientText variant="santaMonica" textSize="3xl">
  Hello World.
</GradientText>
